# 🎮 Pixel About Me

An interactive pixel-art experience where you can explore my room and learn more about me by clicking on different objects.

## ✨ Features

- 🧍 Pixel-art character with idle animation  
- 🖱️ Clickable objects with unique dialogues  
- 🎵 Background music with smooth looping  
- 🔊 Sound effects for interactions  
- 💬 Typewriter text effect  
- 📱 Responsive design (desktop & mobile)  

## 🎯 Concept

This project is designed as a more creative alternative to a traditional "About Me" section.

Instead of reading static text, users interact with a pixel-art room and discover information through objects like:
- 💻 Computer → my interest in programming  
- ☕ Coffee → small personal detail  
- 🎌 Poster → passion for Japan, anime & games  
- 🎮 Console → gaming side  
- 🌇 Window → personal thoughts  
- 🛏️ Bed → personality traits  

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript  

No frameworks, just vanilla code.
